흔한 경험

할 일 앱, 쓰다가 포기한 적 있으신가요?

문제는 의지가 아닙니다. 잊지 않으려고 적어둔 일이 다시 시야 밖으로 밀려나는 구조가 문제입니다.

📋

목록만 쌓이고 줄지 않는다

새 할 일은 계속 추가되지만, 예전에 적은 항목은 아래로 밀려 눈에 잘 띄지 않습니다.

🗑

적어둔 일도 다시 잊힌다

기록은 남아 있지만 다시 보지 않으면, 사실상 안 적은 것과 다르지 않습니다.

🤹

여러 앱을 쓰다 복잡해진다

업무는 노션, 개인은 메모앱, 긴급한 건 카카오톡... 어디에 뭐가 있는지 모릅니다.

🧠

"뭐부터 해야 하지?" 매번 고민

할 일 목록을 열 때마다 무엇을 먼저 할지 결정하는 데 에너지를 씁니다.

적어둔 일
노트나 할 일 앱에 기록했지만 다시 보지 않아 놓치는 일이 생깁니다.
작업 영역
업무, 개인, 집안일처럼 맥락이 다른 일이 한 목록에 섞이면 관리가 흐려집니다.
마무리
완료하거나, 나중으로 넘기거나, 필요 없다고 없애기 전까지 일은 남아 있어야 합니다.
TesQ의 접근법

큐(Queue)로 생각하면 단순해집니다

같은 작업 영역 안에서 완료되지 않은 일은 끝까지 남습니다. 앞선 작업을 마치거나 넘기면, 다음에 다룰 일이 자연스럽게 앞으로 옵니다.

순환 큐 + Tier 시스템

작업은 원형 트랙을 따라 순환합니다. 12시 방향에 도달하면 "지금 해야 할 작업"이 됩니다. 지금 다루기 어렵다면 뒤로 넘길 수 있고, 넘긴 횟수에 따라 Tier 배지 색상이 변합니다.

배지는 불이익이 아니라 상태 표시입니다. Tier Aging으로 오래 방치된 작업도 다시 활성 상태로 돌아올 기회를 얻습니다.

기존 할 일 앱
  • 적어둔 일이 목록 아래로 밀린다
  • 다시 보지 않으면 기록한 일도 잊힌다
  • 목록 아래쪽 일들은 계속 방치된다
  • 뭐부터 할지 매번 스스로 결정해야 한다
TesQ
  • 완료되지 않은 일은 같은 작업 영역에 남는다
  • 앞선 일을 마치면 다음 일이 차례로 보인다
  • 오래 남은 작업은 배지 색으로 알아볼 수 있다
  • 완료·미루기·없애기로 일을 끝까지 다룬다

사용 방법

이렇게 씁니다

복잡한 설정 없이 바로 시작할 수 있습니다.

1

단축키 하나로 언제든 소환

어떤 앱을 사용하든 Ctrl+Shift+D를 누르면 대시보드가 바로 나타납니다. 설정에서 단축키를 변경할 수도 있습니다.

어떤 화면에서든 → Ctrl + Shift + D → 대시보드 등장
2

타이핑으로 할 일 추가

입력창에 할 일을 적으면 끝입니다. #태그로 프로젝트와 카테고리를 분류하고, @now로 즉시 처리, @next로 다음 순서로 배치합니다. 기존 프로젝트/카테고리는 자동완성으로 빠르게 입력할 수 있습니다.

3

지금 다룰 일에 집중

대시보드를 열면 지금 다룰 작업 하나가 가장 크게 보입니다. 먼저 보이는 일을 처리하면 다음 일이 차례로 이어집니다. 제목을 클릭하면 바로 수정할 수도 있습니다.

완료되지 않은 일은 남아 있습니다

지금 당장 하기 어렵다면 미루기로 뒤로 넘길 수 있습니다. 앞선 작업들을 마치거나 넘기다 보면, 아직 끝나지 않은 일은 다시 다룰 차례가 옵니다. 오래 남아 있는 작업은 배지 색으로 한눈에 알아볼 수 있습니다.

궁금한 점들

기존 할 일 앱(노션, 투두이스트 등)과 뭐가 다른가요? ▼
대부분의 할 일 앱은 기록한 목록을 보여주는 데 그칩니다. TesQ는 같은 작업 영역 안의 미완료 일을 끝까지 남겨두고, 지금 다룰 일을 차례로 보여줍니다. 완료하거나, 미루거나, 없애기 전까지 적어둔 일은 시야 밖으로 사라지지 않습니다.
무료인가요? ▼
네. TesQ는 현재 무료 공개 베타로 제공되며, 모든 기능을 제한 없이 사용할 수 있습니다.
데이터는 어디에 저장되나요? ▼
모든 데이터는 내 PC의 로컬 폴더(%APPDATA%\TesQ)에만 저장됩니다. 외부 서버로 전송되지 않으므로 작업 내용이 유출될 걱정이 없습니다.
클라우드 동기화는 되나요? ▼
현재 버전은 클라우드 동기화를 지원하지 않습니다. 대신 인터넷 연결 없이도 항상 빠르게 동작하고, 개인 데이터가 외부로 전송되지 않습니다. 백업 기능으로 로컬 ZIP 파일에 데이터를 안전하게 보관할 수 있습니다.
인터넷 없이도 쓸 수 있나요? ▼
네. 모든 기능이 오프라인에서 동작합니다. 인터넷은 새 버전 업데이트 확인에만 사용되며, 연결이 없어도 앱 사용에는 영향이 없습니다.
Windows 말고 다른 OS에서도 되나요? ▼
현재는 Windows 전용입니다. .NET 8 Desktop Runtime이 설치된 Windows 10 이상에서 사용할 수 있습니다.
개인 정보가 수집되나요? ▼
작업 제목, 메모, 첨부파일 등 개인 작업 내용은 일체 수집하지 않습니다. 앱 버전, OS 버전 등 최소한의 익명 진단 정보만 사용되며, 사용 통계는 별도 동의한 경우에만 수집합니다.
