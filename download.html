설치는 세 단계면 충분합니다.

베타 기간에는 zip 파일로 배포합니다. 설치 프로그램 없이 압축을 풀고 실행하는 방식입니다.

1

다운로드

TesQ.UI.zip을 받은 뒤 원하는 폴더에 저장합니다.

2

압축 해제

zip 파일의 압축을 풀고 폴더 안의 TesQ.UI.exe를 찾습니다.

3

실행

exe를 실행합니다. 이후 업데이트는 앱 안의 안내에 따라 진행할 수 있습니다.

처음 실행 전 알아둘 것

일부 PC에서는 Windows가 실행 전 확인 안내를 보여줄 수 있습니다.

실행에 필요한 구성 요소

앱이 바로 열리지 않고 Microsoft 구성 요소 설치를 안내할 수 있습니다. 안내가 나오면 .NET 8 Desktop Runtime을 설치한 뒤 다시 실행해 주세요.

Windows 확인 안내

처음 받은 앱이라 Windows가 "PC를 보호했습니다" 같은 확인 창을 띄울 수 있습니다. 파일 출처가 TesQ 다운로드 페이지인지 확인한 뒤 추가 정보와 실행을 선택해 주세요.

로컬 데이터

작업 제목, 메모, 첨부파일 내용은 서버로 보내지 않습니다. 워크스페이스 데이터는 사용자 PC의 앱 데이터 폴더에 저장됩니다.

공개 베타

지금은 먼저 써보고 의견을 모으는 공개 베타입니다. 사용 중 불편한 점은 이후 피드백 흐름으로 알려주세요.

권장: 처음 실행할 때는 압축을 푼 폴더 전체를 유지해 주세요. exe만 따로 이동하면 업데이트나 보조 파일 접근이 예상과 다를 수 있습니다.

자주 묻는 질문

다운로드와 실행 과정에서 가장 먼저 마주칠 수 있는 질문만 모았습니다.

네. 지금은 zip 파일을 내려받아 압축을 풀고 실행하는 방식입니다. 실행에 필요한 구성 요소 안내가 나오면 .NET 8 Desktop Runtime을 설치한 뒤 다시 실행해 주세요.
기본 작업 관리는 로컬에서 동작합니다. 다만 최신 버전 확인, 업데이트 다운로드, 사용자가 동의한 통계 전송에는 인터넷 연결이 필요할 수 있습니다.
선택적 사용 통계는 사용자가 동의한 경우에만 전송됩니다. 작업 제목, 메모 내용, 첨부파일 내용, 체크리스트 내용은 수집하지 않습니다.
소개 페이지에서 순환 큐, Tier, 워크스페이스, 알람, 백업, 생산성 분석 등을 더 자세히 볼 수 있습니다.

적어둔 일을 끝까지 다뤄주세요.

다운로드는 여기서, TesQ가 모든 할 일을 잊히지 않게 관리하는 방식은 소개 페이지에서 이어서 볼 수 있습니다.
